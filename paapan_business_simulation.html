3. Detail Model LLM & BYOK (Cost Saver)

Aplikasi Paapan menggunakan Google Gemini. Kunci profitabilitas kita adalah struktur biaya API yang super murah dikombinasikan dengan strategi Bring Your Own Key (BYOK).

🧠 Cost API Gemini (Per 1 Juta Token)

1. Flash-Lite 2.0 Input: $0.10 | Output: $0.40

*Model default u/ Free & Plus. Sangat murah, bisa layani 500+ chat cuma dgn Rp 15.000.

2. Flash 2.5 Input: $0.30 | Output: $2.50

*Untuk Chat Standard & Analisis Gambar. Khusus Plus & Pro.

3. Pro 2.5 Input: $1.25 | Output: $10.00

*Khusus untuk Tier Pro (Query Advanced yang butuh nalar tinggi).

🔑 Strategi BYOK (Bring Your Own Key)

Fitur BYOK kita buka untuk tier Plus dan Pro. Mengapa ini jenius?

  • 0 API Cost: Jika user Plus/Pro memasukkan API Key Gemini mereka sendiri, biaya LLM di server kita turun drastis jadi Nol (Rp 0). Margin bersih 100%.
  • Daya Tarik Developer: Banyak tech-enthusiast punya free credits dari Google. Mereka butuh UI canvas visual (Paapan). Mereka bayar subscription Paapan cuma untuk "menyewa" UI Canvas-nya.
  • Unlimited Feel: User BYOK merasa bebas limit ("unlimited chat"), padahal billing Google jalan di credit card mereka, bukan kita.

4. Struktur Biaya Operasional (Lean Startup)

Biaya ditekan sangat ketat di 6 bulan pertama. Beban terbesar di awal adalah Vercel dan Marketing Konsisten.

  • Marketing (Push Organik): Rp 500.000 / bln (Fix sejak awal)
    Fokus budget bukan Ads/CPC konversi, tapi sebagai pelumas (boost post/giveaway). Strategi utamanya adalah DM 5-10 Kreator/Influencer Edukasi & Produktivitas IG/TikTok tiap hari. Barter akun Pro 6 bulan untuk Review Jujur.
  • Infrastruktur Vercel: $20 (~Rp 320.000) / bln (Fix sejak awal)
    Vercel Pro Day 1. Menghindari limit fungsi serverless dan menjamin loading super cepat untuk kustomer bule/lokal.
  • Database Supabase: Gratis (Bln 1-6) → Rp 400.000 (Bln 7+)
    Kita pakai Free Tier (500MB) selama mungkin karena data vektor (node) relatif ringan. Saat User Total mencapai ±1.500 users (Estimasi Bulan 7), database akan penuh & kita Upgrade ke Pro ($25). Inactivity-Pause diatasi dengan cron job ping otomatis.
  • LLM API (Gemini): Variabel Tinggi Profit
    Biaya sekitar Rp 1.000 - Rp 3.000 per user aktif. Sangat murah berkat Flash-Lite.
Target Utama: Kombinasi kustomer global (margin tebal ~$6 net/user) dan lokal (~Rp 37rb net/user) mempercepat Break-Even-Point. Dengan perbandingan ARPU (Net Revenue Rata-rata) Rp 57.000/User. Untuk mencapai 10 Juta/Bulan, kita hanya butuh ~200 Paying Users.
